Ingé Lumière

Rôle d'un ingé lumière

Les ingé lumière sont responsables de la gestion et installation des lumières lors d’évènements organisés par le BDA. On parle ici des scènes ouvertes, de la soirée BDA, etc… Il se peut que les gens du pôle soirée BDE (et d’autres assos aussi en fait) vous demande de mettre des lumières à leurs évents.

Si vous avez des questions ou des problèmes, n’hésitez pas à contacter Mathieu Bessière qui sera sur Sainté en 2025.

LUMIÈRES

On a deux types de lumières : les spots fixes et les spots mobiles. Il faut bien les brancher (attention il y a parfois des prises qui ne fonctionnent pas) et n’oubliez pas d’allumer tous les interrupteurs derrière le bar.

Spots fixes :
Les régler sur les modes b.001, b.004 et b.007 via le mode A.ddr. Chaque chaîne correspond à R, G ou B.

Spots mobiles :
Manipulez-les avec soin, ils sont déjà abîmés. Scotchez-les pour stabiliser les présets. Réglez-les sur d.013 ou d.019, en mode 6ch (et non 11ch). Une fois allumés, poussez-les vers le haut pour éclairer toute la scène.

INSTALLATION

Pour les ingés lumières, l'installation est la partie la plus délicate. Voici l'ordre des tâches :

  • Rassembler tout le matériel : table de mixage, lumières, câbles DMX, scotch.
  • Brancher la table de mixage et appuyer sur BLACKOUT pour désactiver le clignotement.
  • Fixer la première lumière, la brancher et la mettre sur le bon mode.
  • Relier la table à la lumière avec le câble DMX et tester via la table de mixage.
  • Une fois la première lumière fonctionnelle, enchaîner avec les autres lumières en les reliant entre elles.

TABLE DE MIXAGE

Démarrage :

  • Appuyer sur BLACKOUT si ça clignote.
  • Vérifier que les deux MASTER sont à 10.
  • Mettre le FADE sur 0.

Enregistrement de scènes :

  • Appuyer sur MODE SELECT pour être en 1-24 SIMPLE PRESET.
  • Créer la scène avec les présets A et B.
  • Appuyer sur RECORD seul pour enregistrer.
  • Appuyer ensuite sur RECORD + une touche de scène pour sauvegarder.
  • Attention : cela écrase une scène existante ! Notez bien où vous enregistrez.

Visualisation :

  • Passer en mode SCENE avec MODE SELECT.
  • Tout mettre à 0 sauf la scène que vous voulez voir.
  • Si plusieurs scènes ne sont pas à 0, cela ne fonctionnera pas.
